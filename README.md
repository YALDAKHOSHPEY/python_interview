An advanced, professional-grade terminal-based interview preparation system with multiple quiz modes, intelligent scoring, and comprehensive question bank covering Python, Algorithms, System Design, Databases, and OOP.

## ✨ Features

### Core Features
- 🎯 **Multiple Quiz Modes**: Normal, Mock Interview (timed), Challenge Mode (hard questions), Topic Practice
- 📊 **Advanced Scoring System**: Difficulty multiplier, speed bonus, streak bonus for consecutive correct answers
- 🏆 **Performance Grading**: A+ to F grading system with detailed analytics
- ⏱️ **Timed Questions**: Real interview simulation with configurable time limits
- 🎨 **Colorful CLI**: Professional terminal interface with color coding
- 📈 **Detailed Statistics**: Accuracy tracking, category breakdown, difficulty distribution

### Question Bank Features
- 📚 **50+ Built-in Questions** (easily expandable)
- 🏷️ **Smart Filtering**: By category, difficulty level, tags
- 🔍 **Search Functionality**: Find questions by pattern or tags
- 📊 **Category Support**: Python, Algorithms, System Design, Databases, OOP
- ⭐ **Difficulty Levels**: 1 (Easy) to 5 (Expert)
